File Settings
Enable or disable the creation of backup files.
Enable or disable getting new firmware from FTP server.
Enable or disable downloading settings from FTP server.
Enable or disable uploading settings to FTP server. Filename: -set-yyyymmddhhmmss.ini Timestamp in filename is the file change date on the SD. File transfer only if not present on FTP server. This way only a changed settings.ini file will be placed on the FTP server upon a next transmission.
